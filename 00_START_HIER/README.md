# 00_ZENTRALE_INSEL

Dies ist ab jetzt der zentrale Arbeitsort.

Physischer Sync-Ort:

```text
G:\Meine Ablage\00_ZENTRALE_INSEL
```

Desktop-Einstieg:

```text
C:\Users\derzw\Desktop\00_ZENTRALE_INSEL
```

Grundregel: Alles ausserhalb dieser Insel ist erst einmal Quarantaene oder Quelle. Es wird nichts blind verschoben, nichts geloescht und nichts als "funktioniert" markiert, bis es in der Insel einen Manifest-Eintrag, einen Test und einen sichtbaren Status gibt.

## Ordnerlogik

- `00_START_HIER`: kurze Orientierung, Startplaene, Tagesbrief.
- `01_INGEST_INBOX`: kontrollierte Eingangsschleuse fuer neue Dateien.
- `02_QUARANTAENE_ALTSYSTEM`: Verweise und spaetere sichere Kopien alter Systemteile.
- `03_MANIFESTE_INVENTAR`: maschinenlesbare Inventare, Dateilisten, Klassifizierungen.
- `04_REPORTS_AUDITS`: menschenlesbare Audits und Entscheidungen.
- `05_RAG_SOURCE_OF_TRUTH`: nur gepruefte Wissensquellen, nicht Rohchaos.
- `06_GATEWAY_LIVEFEED`: Gateway, Livefeed, Monitoring, Browser-Capture.
- `07_3D_ASSET_LIBRARY`: getrennte 3D-Bibliothek fuer Staedte, Raeume, Charaktere, Props, Texturen.
- `08_TOOLS_SCRIPTS`: kleine, gepruefte Werkzeuge fuer Ingest, Klassifikation, Status.
- `09_ARCHIV_NICHT_ANFASSEN`: bewusst konservierte Artefakte.
- `99_TMP_NUR_KURZLEBIG`: temporaer, darf nie Quelle der Wahrheit werden.

## Sofortregel

Wenn ein Tool, Agent oder Script etwas erzeugt, landet es nicht mehr irgendwo in `Downloads`, `Sovereign`, `Base`, `.codex`, `.gemini` oder `Desktop` verstreut. Es muss entweder in die Insel, oder als Quellfund mit Pfad ins Manifest.
